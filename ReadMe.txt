Выгрузка данных из системы автоматического учета рабочего времени CrocoTime через Web API.

Инструкция по запуску скрипта для выгрузки активности из Crocotime в csv-файлы.

    1. В файле config.py измнить необходимые настроки:
    1.1 В переменной CT_PARAMS изменить значения 'url', 'token' и 'app_version'.
    'url' - url адрес-подключения к crocotime. Можно скопировать из браузера (с указанием протокола: http или https).
    'token' - сгенерированный токен из крокотайма (Конфигурация -> Токены доступа).
    'app_version' - версия крокотайма (можно посмотреть на странице Конфигурация).

    1.2. Указать период выгрузки.
    Переменные DATETIME_BEGIN и DATETIME_END. 
    Дата указывается в формате "гггг-мм-дд ЧЧ:ММ:СС"
    При необходимости можно изменить формат даты (см. доку по форматам http://strftime.org/)

    1.3. Указать, нужно ли делать посекундную выгрузку.
    Переменная SPLIT_PER_SECONDS. Присвоить значение True (регистр имеет значение), если необходима посекундная выгрузка.
    ВНИМАНИЕ! При включении данной опции может привести к значительному увеличению времени работы скрипта, особенно если данные выгружаются за длительный период или по многим сотрудникам. Например, за 1 день 715 сотрудников могут генерировать 450000 строк в таблице window_switch_train.

    1.4. Если это необходимо, указать путь к папке, куда будут выгружены файлы.
    Переменная PATH. Указать абсолютный путь к необходимой папке. Например, PATH = r'C:\crocotime_files\'. Если путь будет указан неправильно, или такой папки не существует, то данные выгрузятся в путь по умолчанию (...\cctm2csv\export_files). После выгрузки скрипт напишет, куда были выгружены файлы.
